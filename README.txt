Blackjack

Author: Phong Van Nguyen

Program's description: This program simulates a blackjack game in Java.

How To Run:

***Unzip the folder and use one of the following methods.***

(1) Use the jar file. Then, use this command java -jar /path-to-jar-file/BlackJack.jar.
(2) Import into Eclipse or InteliJ and Run.
